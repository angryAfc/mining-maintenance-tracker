# Mining Operations Issue & Maintenance Tracker

## Project Overview

Mining Operations Issue & Maintenance Tracker is a personal software engineering project designed to simulate how mining operations teams track equipment issues, maintenance tasks, and operational risks.

The project allows users to view equipment, report maintenance issues, update issue status, and monitor high-priority operational problems through a dashboard.

## Purpose

This project is being developed to demonstrate practical software engineering skills, including frontend development, API design, automated testing, technical documentation, and real-world problem solving.

## Planned Tech Stack

- Frontend: React, TypeScript
- Backend: C# .NET Web API
- Database: SQLite or PostgreSQL
- Scoring Logic: Python
- Testing: Vitest, React Testing Library, xUnit, pytest
- Documentation: Scoping document, test plan, API documentation, as-built documentation

## Current Status

Currently developing the frontend prototype using React and TypeScript with mock mining equipment and issue data.
