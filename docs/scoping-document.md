# Scoping Document

## Problem

Mining operations rely on many types of equipment, such as haul trucks, conveyor belts, drill rigs, and crushers. Maintenance teams need a clear way to track equipment issues, prioritise risks, and monitor repair progress.

## Goal

The goal of this project is to build a web-based issue and maintenance tracking system that allows users to manage equipment issues and view operational maintenance status.

## Target Users

- Maintenance coordinators
- Site supervisors
- Operations support teams
- Software engineering portfolio reviewers

## In Scope

- Equipment list
- Issue tracker
- Issue creation form
- Status update
- Dashboard summary
- Priority scoring
- Basic automated tests
- Technical documentation

## Out of Scope for MVP

- User authentication
- Real-time IoT sensor data
- Mobile app version
- Production deployment
- Role-based access control

## Success Criteria

The MVP is considered successful when users can view equipment, create issues, assign severity, update status, and view a dashboard summary.
